
# Fenix Multi-Tenant Commerce Backend

## Tech
- Java 17
- Spring Boot 3
- Spring Data JPA
- MySQL
- Maven

## How to run
1. Create MySQL DB using provided `fenix_test_db.sql`
2. Update credentials in `application.yml`
3. mvn clean spring-boot:run

## APIs implemented
- Order CRUD
- Fulfillment CRUD

Aligned with provided OpenAPI 3.0 specification.
